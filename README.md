# DM Tool

DM Tool is a single-file browser app for tabletop encounter management. It starts as an initiative tracker, then switches into an HP tracking view for NPCs and combatants, with import and export support for the full encounter state.

## Features

- Add combatants with manual initiative values or auto-rolled d20 initiative
- Support initiative modifiers plus advantage or disadvantage rolls
- Sort and step through turn order during the encounter
- Track damage, healing, temporary HP, and per-NPC history
- Edit combatants during an encounter
- Export and import encounter data as JSON

## Running Locally

There is no build step or backend required. Open `index.html` directly in your browser.

If you prefer, you can also serve it from any static file server.
